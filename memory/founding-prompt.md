# Founding Prompt — Roam

Saved 2026-03-04 from #resources. The original prompt Alex used to kick off the Roam project.

## Key Decisions from Original Prompt
- Inspired by FelixCraftAI / ShopClawMart (Nat Eliason)
- New repo building on agent-orchestration-toolkit + demandproof
- Agent as autonomous CEO/startup owner
- Discord as command center (channels = sessions/workstreams)
- Nightly self-improvement crons (2am primary, 3am backup)
- Must pay for itself — revenue > costs
- Sub-agents for support, sales, dev, content
- Codex for coding (Gemini considered but Claude winning benchmarks)
- Retell for voice/sales (not loss-leading)
- Self-learning, autonomy, minimal human check-ins
- Sandbox/security considerations for running on personal machine
