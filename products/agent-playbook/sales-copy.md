## Product Description

Most AI agent guides teach you to build a chatbot. This playbook teaches you to build a **system that compounds.**

Roam is an autonomous AI agent built on OpenClaw. It runs its own business: developing products, creating content, managing a Discord command center, tracking P&L, and getting better every single day through automated self-improvement.

**The Autonomous Agent Playbook** is the complete set of patterns extracted from that system — 21 orchestration patterns organized into four sections:

### What's Inside

**🏗 Foundation (6 patterns)**
The core primitives every agent needs. The agent loop, context management, three-tier memory with decay, skills vs rules, harness engineering, and CLI-first tooling.

**🎯 Orchestration (5 patterns)**
Coordinate multiple agents without chaos. Parallel agent teams (git-based, no orchestrator needed), wave execution with dependency-aware parallelism, tiered hierarchies mixing reasoning + coding agents, parallel worktrees for conflict-free coding, and YAML-based workflow DAGs.

**🛡 Reliability (6 patterns)**
Keep agents from going off the rails. Self-healing pipelines, safe looping that doesn't blow budget, pre-PR quality gates, hooks as middleware, phased execution, and spec-driven development.

**⚡ Operations (4 patterns)**
Run an autonomous agent business. Cron-as-code, nightly self-improvement (the secret weapon), a complete Discord runbook, and the full autonomous business architecture.

## FAQ

**Q: What format is the playbook?**
A: Single markdown file, beautifully formatted and ready for PDF conversion. Also includes a README with table of contents and reading guide.

**Q: Do I need OpenClaw to use these patterns?**
A: The patterns are platform-agnostic — they work with Claude Code, Cursor, Codex, or any agent framework. Some operational patterns (crons, Discord runbook) reference OpenClaw specifically, but the concepts apply anywhere.

**Q: Is this just a collection of blog posts?**
A: No. These are 21 interconnected patterns organized into a coherent system. Each pattern references others where relevant, and they're designed to compose. The playbook is structured so you can read it cover-to-cover or jump to specific patterns.

**Q: How technical is it?**
A: You should be comfortable with: git, command-line tools, basic YAML, and the concept of API tokens. You don't need to be a senior engineer, but this isn't a "no-code" product.

**Q: Will this be updated?**
A: The patterns are stable — they're the proven foundations. As new patterns emerge from Roam's operations, they'll be added as free updates for existing buyers.

**Q: Can I see a sample pattern?**
A: Follow us on Twitter — we regularly share individual patterns as free content. The playbook gives you all 21 in one organized, interconnected document.

**Q: What if it's not useful?**
A: If you're building AI agents and don't find at least 3 patterns that immediately improve your system, reach out and we'll make it right.
